Explorations in Dynamic Iconography

v.1 :: Basic Kinematic Relationship

VIEW

The general exploration direction for this prototype was to create a basic kinematic relationship between graphical objects. This activity involved connecting virtual bone and joint components to form a kinematic structure called an armature.

v.2 :: Passive Kinematic Figure

VIEW

The general exploration direction for this prototype was to create a passive kinematic figure. This activity began by analyzing the chosen model, Helvetica Man (Lupton & Miller, 1996), and then progressed to building and combining multiple armatures representative of the model’s form.

v.3.1 :: Interactive Kinematic Figure

VIEW

The general exploration direction for this prototype was to create an interactive kinematic figure capable of responding to two commands, a wave and a kick. This activity involved developing a user interface and programming animations for the figure. The animations are elicited by typing “wave” or “kick” into the prototype’s textfield and pressing ENTER.

v.4.1 :: Role-Play Selection Icons

VIEW

The general exploration direction for this prototype was to create two dynamic icons that included rich media (e.g., animation, sound). The proposed functions of these icons were to allow a user to 1) indicate a fictional identity’s status, and 2) choose an identity for his or herself to role-play. This activity required developing a user interface and programming the rich media feedback of the dynamic icons.

v.5.1 :: Targeted Messages

VIEW

The general exploration direction for this prototype was to create a third dynamic icon that communicated targeted messages using rich media, and integrate it with the icons developed in the previous prototype. The determination of which message the new icon communicates is based on a user’s choices with the other icons. This activity involved further developing the previous prototype and programming the rich media feedback of the new dynamic icon.

Lupton, E. & Miller, A. (1996). Design, writing, research: Writing on Graphic design. London: Phaidon.
